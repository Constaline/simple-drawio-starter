# simple-drawio

Basic Drawio for NeteaseCloudMusic.

## 开发

### 启动方式

```bash
# 安装 serve
sudo npm install -g serve

# 在项目根目录
serve
```
> 启动完成后，项目默认运行在 http://localhost:5000
